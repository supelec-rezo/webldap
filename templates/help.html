Aide

webldap est l'application de gestion des comptes LDAP FedeRez. Cette version de webldap est un fork de la version de FedeRez, adaptée aux besoins de Supélec Rézo.

Utilisateurs

Les utilisateurs sont les membres de Supélec Rézo. Si vous n'avez pas de compte, vous pouvez en demander un au bureau de l'association ou aux administrateurs du LDAP.

À propos

Les sources de l'application sont publiques :

  • dépôt Github de webldap (Supélec Rézo)
  • dépôt Github de webldap (FedeRez)
